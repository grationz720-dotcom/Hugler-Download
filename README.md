# Hugler Download 🚀
**The Ultimate Auto-Updater and Library Sync Tool for Python Developers.**

## Features
- **Auto-Sync:** Automatically installs missing libraries from GitHub config.
- **Project Info:** Displays links, social media, and project name in the terminal.
- **Developer Tools:** Easy setup for your own projects via `config.json`.
- **Self-Update:** Keeps the Hugler Download core up to date.

## How to use
1. Create a `config.json` on your GitHub.
2. Link it to Hugler Download.
3. Your users will always have the latest version!
